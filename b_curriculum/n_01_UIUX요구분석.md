##### 문서의 메모와, 그를 업데이트(버전관리) 시스템이해

- 마크다운 사용법 - `*.md`
- 깃사용법(계정), 내 폴더관리 - 보내기(git push), 받기(git pull)
  - `git add 선택 ` - 선택(생성, 수정, 삭제)
  - `git commit -m "메모"`  - 메모
  - `git push` - 보내기
  - `git pull` - 받기
  - 만약, 컴퓨터에  기존 git자료가 없다면 `git clone 주소`  으로 자료를 동기화(받기)
